A static frontend clone of the Amazon homepage built specifically to demonstrate **responsive web design** and **advanced CSS layout techniques** (Flexbox & Grid). This project focuses on mimicking the visual interface and user experience of Amazon.com without backend functionality.

## üìö Prerequisite Knowledge

To understand and modify the source code of this project, you should be familiar with the following concepts:

* **HTML5 Structure:**
    * Semantic tags (header, nav, section, footer).
    * Forms and input elements (for the search bar).
    * Image handling (`img` tags and paths).
* **CSS3 Styling:**
    * **Flexbox:** Used heavily for the navigation bar and product rows.
    * **CSS Grid:** Used for the main product layout gallery.
    * **Media Queries:** Used to make the website responsive (mobile vs. desktop view).
    * **Box Model:** Margins, padding, and borders.
* **Version Control (Optional):**
    * Basic Git commands (`git clone`, `git add`, `git commit`).

## ‚ú® Key Features

This project recreates the visual elements of Amazon, including:

* **Responsive Navigation Bar:** Complete with search bar, location, and menu icons.
* **Hero Section:** Replicates the Amazon carousel/banner background style.
* **Product Grid:** A responsive card layout using **CSS Grid** that adjusts columns based on screen size.
* **Hover Effects:** Interactive buttons, product cards, and links that react to mouse events.
* **Footer:** A detailed, multi-column footer replicating the sitemap links.
* **Mobile Responsiveness:** Uses media queries to adapt the layout for tablets and mobile devices.


## üõ† Tech Stack

* **HTML5:** Semantic structure (header, main, section, footer).
* **CSS3:**
    * **Flexbox:** Used for the navigation bar and aligning items within rows.
    * **CSS Grid:** Used for the main product layout structure.
    * **Media Queries:** For making the website fully responsive.
* **FontAwesome:** For search icons, cart icons, and menu bars.

## ‚öôÔ∏è How to Run the Project

Since this is a static website, you don't need to install any dependencies.

**Method 1: Direct File Open**
1.  Download or Clone the repository.
2.  Navigate to the project folder.
3.  Double-click the `index.html` file.
4.  The website will open in your default web browser.

**Method 2: VS Code Live Server (Best Experience)**
If you are using VS Code:
1.  Install the "Live Server" extension.
2.  Right-click on `index.html`.
3.  Select **"Open with Live Server"**.
